Hidden 隐藏域 ​

改值体现于最终提交的表单中，用于例如编辑记录的id这种场景中

TS 定义 ​

typescript
interface Hidden extends FormItem {
  type: 'hidden';
}

点击查看FormItem的定义

基础用法 ​

在开启多选模式后，默认情况下会展示所有已选中的选项的Tag 

[
  {
    type: "hidden",
    name: "hidden"
  }
]
