use ​

使用中间件的方式扩展方法，上述方法中标记有扩展支持： 是的方法都支持使用use扩展

usePlugin ​

  • 详情：

相对于use, usePlugin支持更加灵活更加细致的扩展， 上述方法中标记有扩展支持： 是的方法都支持使用usePlugin扩展

每个支持扩展的方法都支持定制before、after两个hook来干预原有方法的行为，before可以用于修改传入参数，after可以用于修改返回的值
