# Textarea 文本域

# 基础用法

在开启多选模式后，默认情况下会展示所有已选中的选项的Tag 

[
  {
    type: "textarea",
    name: "textarea",
    text: "文本域"
  }
]
显示配置

# 禁用状态

通过 disabled 属性指定是否禁用 input 组件 

[
  {
    type: "textarea",
    name: "textarea",
    text: "文本域",
    disabled: () => true
  }
]
显示配置
