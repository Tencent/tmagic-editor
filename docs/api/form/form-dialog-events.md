# FormDialog组件 events

## close

- **详情：** 弹出关闭

- **回调函数：** () => void


## submit

- **详情：** 调用[save()](./form-dialog-methods.md#save)

- **回调函数：** (values: any) => void

## error

- **详情：** 表单校验不同

- **回调函数：** (error: any) => void

## change

- **详情：** 表单中任何值发生变化

- **回调函数：** (values: any) => void
